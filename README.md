# STM32_WS2812B
![img6](https://github.com/Nocturn42/STM32_WS2812B/assets/112398152/bd36cd74-7ece-4999-9e19-66a26267c910)
### 透過STM32上面的藍芽模組HC-05與手機進行連接，將手機的信號發送到單晶片上，使LED模組WS2812B能夠按照手機的指示進行發光
所設計的發光模式包含恆亮、呼吸燈、流星燈以及彩虹循環四種。
